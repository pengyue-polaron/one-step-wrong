# Asset Notes

This file records the origin and intended use of committed media so the repository can be reviewed without guessing about third-party rights.

## Product Screenshots

Files in `artifacts/screenshots/` are captures of this repository running locally. Playwright produces the maintained flow screenshots from reviewed test states. They contain fictional interface content and no learner data, credentials, or private campus material.

## Voice Note

`public/audio/the-voice-you-know-opening.ogg` is a short synthetic performance created for this project. The speaker and script are fictional. It is not a clone of a real person and is not generated at runtime.

## Illustration

`public/pixel-study-room.svg` is an original repository asset used by the legacy Final Submission chapter. It does not reproduce an NYU logo or official campus artwork.

## Icons And Product Marks

Interface icons come from `lucide-react`, which is distributed under the ISC License. One Step Wrong uses its own text lockup and numbered square mark. The repository does not include official NYU, Brightspace, Duo, Google, or OpenAI logo assets.

Real service and institution names appear only where needed to describe sourced public guidance or the setting of a legacy teaching case. Reviewed learner-facing rehearsals publish under the fictional Northbridge University setting by default.
