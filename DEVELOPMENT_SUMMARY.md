# 多章节产品开发总结

## 已完成

- 完整可玩的「最后一次提交」vertical slice，覆盖四种网络路径
- 官方认证、访客认证、个人热点与假接入/配置流程
- 危险路线中的真实提交成功、操作驱动的延迟异常与通知归档
- 可独立执行的会话退出、密码修改、MFA、网络断开/忘记、配置删除、消息删除、同学提醒和 IT 工单
- 三个动态结局、三项复盘评分、事件时间线、实际线索、有效止损行为与关键节点重玩
- 统一像素桌面、夜间学习空间资产、窗口系统、响应式桌面布局、音效静音、动画减弱和倒计时暂停
- Scenario 注册、配置注入、集中式 reducer、稳定事件 ID 和完整检查点重置
- 状态机、案例组件测试与九条 Playwright 完整流程
- NYU Bobst Library 故事背景、Brightspace 作业界面与 NYU Violet 品牌层
- 复盘中的因果链与网络名、根域名、权限边界、分层止损四项迁移检查
- 案例档案库、会话内完成进度和统一章节导航
- 「共享范围」NYU Drive 章节：三种授权范围、外部访问后果、版本恢复与小组沟通
- 「这次是你吗」Duo 章节：意外推送判断、未知会话后果、恢复方式检查与安全上报
- 数据驱动 `DecisionCaseDefinition` 与通用 intro / decision / outcome / response / debrief 运行器

## 未完成

当前三章产品范围内无已知未完成项。背景音乐、窗口拖拽、跨会话进度和多语言属于后续扩展，不影响当前产品体验。

## 已知问题

- 宽度低于 1100 px 不提供完整交互，只显示桌面浏览器提示
- 窗口使用固定布局，不支持拖拽
- Web Audio 在浏览器禁止音频时会静默降级，不影响操作

## 下一阶段建议

1. 使用相同章节运行器增加钓鱼邮件、二维码登录或公共充电设备案例。
2. 为 scenario 增加通用 effect 解释器，进一步减少 reducer 中的案例专有 action。
3. 在学校 IT 展示反馈稳定后，再评估背景音乐、窗口拖拽与多语言支持。
